# 💈 Barber System - Appointment Management App

## 📌 Overview
Barber System is a web application designed to help barbershops and small businesses manage appointments, clients, and schedules بسهولة and efficiently.

Many barbers still manage bookings through WhatsApp or paper, which leads to missed appointments and disorganization. This system solves that problem.

- 📅 Appointment scheduling
- 👥 Client management

- 🔥 Real-time data with Firebase

- 🔐 Authentication (if implemented)

## 🧠 Problem it solves

Small businesses like barbershops often struggle with:

- Lack of organization

This app provides a simple and scalable solution to manage daily operations.
